OTP Bot Extension

Automatically capture and manage OTP codes from any webpage

Download OTP Bot Extension

Installation Steps

  1. Download and extract the otp-bot-extension.zip file
  2. Open Chrome and navigate to chrome://extensions
  3. Enable "Developer mode" in the top right corner
  4. Click "Load unpacked" and select the extracted folder
  5. The OTP Bot extension will appear in your Chrome toolbar

Features

  • Automatic OTP detection (4-8 digits)
  • Modern, clean interface
  • One-click copy to clipboard
  • Timestamp and source URL tracking
  • Stores up to 10 recent OTPs
